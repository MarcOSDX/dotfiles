# About

Backup of my UNIX configuration files, I try keep all configuration settings by minimalist way.

## Requirements

* An UNIX system or GNU/Linux system
* git 2.24.3+
* stow 2.3.1+

## Gettings dotfiles configuration

```bash
user@localhost:~> git clone --recursive https://github.com/marcos-ro/dotfiles ~/.dotfiles
```
